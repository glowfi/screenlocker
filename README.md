# DEPENDENCIES

-   Python
-   Rust

# INSTALL

```
pip install opencv-python tk pynput playsound pathlib pyautogui
git clone https://github.com/glowfi/screenlocker
cd screenlocker
cargo build
mv ./target/debug/screenlocker ~/.local/bin/screenlocker
cd ..
rm -rf screenlocker
```
